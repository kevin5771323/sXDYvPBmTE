# 宠物领养管理系统 java4

## 项目概述

宠物领养管理系统是一个基于B/S架构和MVC模式的宠物领养信息管理平台。系统采用了前后端分离的设计，前端使用Vue.js框架，后端采用Java和SpringBoot技术栈。

## 技术栈

- 架构：B/S、MVC
- 系统环境：Windows/Mac
- 开发环境：
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- 技术栈：
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 系统角色

#### 管理员

- 前台操作：
  - 宠物领养信息管理
  - 用户反馈处理等
- 后台管理：
  - 用户管理
  - 角色管理
  - 菜单管理
  - 文件管理
  - 流浪动物管理
  - 待绝育动物管理
  - 申请领养管理
  - 评论管理
  - 流浪动物救助
  - 喂养点管理
  - 动物走失管理
  - 救助站管理
  - 帖子管理
  - 捐赠管理
  - 公告管理
  - 科普文章管理
  - 活动管理

#### 用户

- 首页浏览
- 宠物领养申请
- 流浪宠物救助参与
- 宠物喂养点查询
- 丢失宠物信息查看
- 流浪宠物救助站信息查询
- 宠物论坛互动
- 宠物捐赠活动
- 科普文章阅读
- 热门活动参与

## 运行环境

系统可运行在以下环境中：

- 服务器：推荐使用支持Java和Mysql的服务器
- 客户端：任何支持现代Web浏览器的操作系统
